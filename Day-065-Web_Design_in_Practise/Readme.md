# Day 65 – Hotel Landing Page with Canva

## Project Overview

This is a visually stunning hotel landing page website designed using Canva, a graphic design platform. The project focuses on applying core web design principles to create an engaging and user-friendly promotional page for a hotel. The design incorporates thoughtful color schemes, readable typography, intuitive user interface elements, and considers user experience to create a page that effectively communicates the hotel's brand and offerings. While not coded from scratch, this project demonstrates an understanding of how design principles translate into effective web layouts that can later be implemented with HTML, CSS, and JavaScript.

## What I Have Learned

* **Introduction to Web Design**: Learned that web design is the process of planning, conceptualizing, and arranging content intended for the internet. It goes beyond just making things look pretty—it's about creating functional, accessible, and engaging experiences that help users achieve their goals. Good web design considers both aesthetics and usability to keep visitors engaged and guide them toward desired actions.

* **Color Theory**: Color theory is the study of how colors interact and the emotional responses they evoke. Different colors trigger different feelings blue feels trustworthy and calm (why banks use it), red creates urgency (sale tags), green feels natural and peaceful, yellow grabs attention, and purple feels luxurious. Good color choices make a website visually appealing, establish brand identity, guide users to important elements, and improve readability through proper contrast. Using colors that work well together creates harmony, while poor color choices can make a site feel unprofessional or even inaccessible to colorblind users.

* **User Interface (UI) Design**: UI design focuses on the visual elements users interact with like buttons, icons, menus, forms, images, spacing, and layout. Good UI design makes a website look polished and professional while being intuitive to use. Every element should have a clear purpose and appear in a logical place. For example, buttons should look clickable (using shadows, borders, or colors), forms should be easy to fill out, and navigation menus should be easy to find. UI design answers the question "What does the user see and click on?"

* **Typography**: Typography is the art of arranging text to make it readable, legible, and visually appealing. It involves choosing fonts, sizes, line spacing, letter spacing, and text alignment. Good typography creates a clear hierarchy (headings are bigger than body text, so users know what's important), improves readability (proper line height makes long paragraphs easier to read), and sets the tone of the website (serif fonts feel traditional and elegant, sans-serif feels modern and clean). Without good typography, even great content can feel overwhelming or difficult to read.

* **User Experience (UX) Design**: UX design focuses on how users feel when interacting with a website and how easily they can accomplish their goals. It considers the entire journey from first arriving on the site to completing an action like booking a room or buying a product. Good UX means intuitive navigation (users never wonder "where do I click next?"), fast loading times, clear information, minimal steps to complete tasks, and accessibility for all users including those with disabilities. UX design answers the question "How does the user feel and how easy is this to use?" Unlike UI which is about visual elements, UX is about the overall experience.
