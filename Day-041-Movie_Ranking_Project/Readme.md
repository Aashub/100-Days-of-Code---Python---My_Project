## What I Have Learned

* **HTML Headings**: Learned how to use heading tags like <h1></h1>, <h2></h2>, and <h3></h3> to structure content properly and what is hierarchy to use them.

* **Paragraph Tag**: : Learned about how to use the <p> tag to write descriptions and keep text readable.

* **Horizontal Rule**: Learned how to use <hr /> horizontal rule to visually separate different sections on the webpage.

* **Break Tag**: Learned how to use <br/> break tag to break line and spacing between elements.

* **Void Tag**: Learned that some HTML tags do not have closing tags because in this tag their is no need to add content therefore do not require a closing tag and that's why they known as void tags, such as <br /> and <hr />.

* **Non Void Tag**: Learned that most HTML tags come with opening and closing tags, known as non-void tags, such as <h1></h1> and <p></p>.
